黑群辉洗白方案-7.X版本

  2023-02-05   NAS   黑群辉 字数统计: 1293字 阅读时长: 3分

0x01 前言

　　买了个黑群辉不洗白有些功能用不了，买个洗白码也用不了多少钱，所以就有了此篇文章

0x02 全黑群晖/半洗白/全洗白 有什么区别

2.1 半洗白和全洗白需要什么条件

　　全洗白：SN码和mac地址都正确(一般需要去淘宝或者闲鱼购买，不同的型号价格不一样，一般10-20元)

　　半洗白：SN码正确，mac地址对不上(网上一般可找到sn码子，网上花点时间找找即可，白嫖)

3.2 全黑群晖/ 半洗白 /全洗白 有什么区别

　　1、没洗白：

　　⑴ 不能注册和登录QuickConnect（简称“QC”，就是没有公网的用户利用群晖服务器进行内网穿透）；

　　⑵ DS Photo/Moments套件的预览图是黑的；

　　⑶ DS Video看片时视频质量的高、中、低码率不能自由切换只能选择原码；

　　⑷ DS Video不能离线转码（群晖默认不能播放RM/RMVB等文件格式，以前网上下载的影片多为此类格式，所以有时需要转换成能播放的MP4文件格式）。

　　2、半洗白：只需要正确的序列号即可,mac 地址不合法也无所谓,可以使用视频的缩略图和转码功能,但是无法使用 qc 功能

　　⑴ DS Photo/Moments预览图正常；

　　⑵ DS Video的视频质量高、中、低码率可以自由切换；

　　⑶ DS Video可以离线转码。但是不能注册和登录QuickConnect。

　　3、全洗白：以上的限制全部都没有啦包括登录QuickConnect。只有全洗白，才能够正常的登录QC，群晖的所有功能都可以使用(dsm7会检测qc或者ddns，不建议使用该功能)

　　​​

　　‍

　　‍

　　‍

0x03 简易洗白方法

3.1 修改ip地址

　　登录群晖后台–控制面板–网络–网络界面–点击编辑

　　​​​

　　‍

　　改成自动获取网络设备，如下图（此操作是为了避免mac地址变动造成群晖局域网IP变更【如果路由器后台设置mac-IP地址绑定的请先取消绑定】）

　　​​

　　‍

3.2 开启ssh

　　开启SSH，端口默认22即可：【洗白完成后即可关闭ssh】；

　　​​​

　　‍

3.3 登录ssh并切户root用户

　　‍

　　登录ssh，输入群晖管理账号回车，密码回车【输入的密码是看不见的】，输入 sudo -i ​​ 回车，再次输入密码回车。整个流程如图；

　　​​

　　‍

3.4 挂载boot目录

　　‍

　　复制下面的代码,鼠标进入putty窗口后[右键粘贴]再回车，回车后画面如图；

mkdir -p /tmp/boot 
cd /dev 
mount -t vfat synoboot1 /tmp/boot/ 
cd /tmp/boot/boot 
cd grub 
vi grub.cfg

　　​​

3.5 修改SN和MAC地址

　　方向键进入最下方2大段内容修改sn、mac1；务必注意不要修改其他内容，sn mac前后都有一个空格。

　　​​​​

　　进入群晖网页后台，重启下群晖系统洗白流程完毕。在重启的过程中中注册个账号https://account.synology.cn/zh-cn/profile​

　　‍

3.6 验证洗白功能

　　注册成功之后登录群辉，在设置中登录登录账号，解锁全部功能

　　​​

　　‍

　　在套件中心安装Active Backup for Business ​​ Synology Photos​ Advanced Media Extensions​三个套件是否正常，打开Advanced Media Extensions​，检查解码是不是正常

　　​​

　　‍

　　‍

0x04 其他洗白方法

　　如果不想用上述方法洗白的，先将群辉关机，将启动盘插入电脑，使用DiskGenius​找到U盘，找到grub.cfg文件，修改里面的SN码和MAC地址

　　​​​​

　　‍

Apache Log4j任意代码执行漏洞

  2021-12-10   漏洞复现   漏洞复现 字数统计: 919字 阅读时长: 2分

0x01 漏洞描述

Apache Log4j存在任意代码执行漏洞，经过分析，该组件存在Java JNDI注入漏洞，当程序将用户输入的数据进行日志，即可触发此漏洞，成功利用此漏洞可以在目标服务器上执行任意代码。

0x02 影响范围

Apache Log4j 2.x < 2.15.0-rc2

不受影响版本
Apache Log4j 2.15.0-rc2（与官网的2.15.0稳定版相同）
Apache Log4j 2.15.1-rc1
